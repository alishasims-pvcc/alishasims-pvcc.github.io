    Operating Systems Terminology

Operating System Software that conntrols the overall operation of a computer. It provides the means by which a user can store ans retrieve files, provides the interface by whick a user can request the execution of programs, and provides the environment necessary to execute the programs requested.
Job Queue In batch processing systems, the jobs residing in mass storage waiting for execution.
FIFO Jobs in the queue, or storage organization, that are ordered in first-in, first-out (abbreviated FIFO) fashion.
Interactive Processing New operating systems developed that allow a program being executed to carry on a dialogue with the user through remote terminals.
Time-Sharing Operating systems that provide service to multiple users at the same time.
Multiprogramming A technique in which time is divided into intervals and then the execution of each job is restricted to only one interval at a time.
Application Software Consists of the programs for performing tasks particular to the machine's utilization.
System Software Perfoms those tasks that are common to computer systems ingeneral. Provides the infrastructure that the application software requires.
Utility Software Consists of software units that extend (or perhaps customize) the capabilities of the operating system.
Graphical User Interface (GUI) Tasks performed by more modern systems in which objects to be manipulated, such as files and programs, are represented pictorially on the display as icons.
Shell An older user interface. A component within the domain of an operating system that handles the communication between the computers user and operating system.
Kernel The internal part of an operating system.
Device Driver Another component of the kernal. The software units that communicate with the controller (or at times, directly with peripheral devices) to carry out operations on the peripheral devices attached to the machine.
File Manager Software component within the kernal, whose job is to coordinate the use of the machine's mass storage facilities.
Memory Manager Component of the operating system's kernel, coordinates the machine's use of main memory.
Virtual Memory Large "fictional" memory space created by paging.
boot memory The procedure that transfers the operating system from mass storage (where it's permanently stored) into main memory (which is essentially empty when the machine is first turned on).
ROM (Read-only memory) A small portion of a computer's main memory where the CPU expects to find its initial program is constructed from special nonvolatile memory cells. Contents can be read but not altered.
Deadlock The condition in which two or more processes are blocked from progressing because each is waiting for a resource that is allocated to another.
Spooling The technique of holding data for output at a later but more convenient time.
Super User (or Administrator) Person that estabilishes accounts. Gains highly privileged access to the operation system.
Multi-Factor Authentication More than one piece of evidence to gain access to secured computer accounts.
