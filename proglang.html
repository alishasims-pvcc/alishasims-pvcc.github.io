Item Definition
Machine Language Programs for modern computers consisting of sequences of instructions that are encoded as numeric digits.
Debugging Errors that must be located and corrected when writing programs in a machine language.
Assembler Developed to convert mnemonic expressions into machine language instructons.
Assembly Language A mnemonic system for representing programs.
Translator A program written that translates programs expressed in the high-level primitives into machine-language into machine-language programs.
Compiler Translation programs.
Interpreter An alternative to translators, another means of implementing third-generation languages.
Cross-Platform Software Software that is independent of an operating system's design as well as the machine's hardware design and is therefore executable throughout a network.
Natural Language Languages such as English, German, and Latin.
Formal Language Programming languages.
Machine Independence Programs to be transferred and executed across networks and internets involving different machine designs and different operating systems.
High-Level Programming Language Provides more abstractions for the programmer and makes it easier for people to read and write a program.
Object-Oriented Programming (OOP) Programming process where a software system is viewed as a collection of units, called objects, each of which is capeble of performing the actions that are immdeiately related to itself as well as requestion actions of other objects. Together, these objects interact to solve the problem at hand.
Scripting Language A subset of the imperative programming languages.
Data Types
  • Integer: Refers to numeric data consisting of whole numbers, probably stored using two's complement notation.
  • Float: Sometimes called "Real", refers to numeric data that might contain values other that whole numbers, probably stored in floating-point notation.
